# Euphoric v4.2

A peaceful Chrome new-tab extension with daily scenery, quotes, tasks, and a logo-only animated butterfly reminder companion.

Load this folder in Chrome using Developer Mode.

For the desktop butterfly reminder, install the companion from:

`RUN_THIS_FOR_LOGO_BUTTERFLY_COMPANION_windows`


v6.5: Date validation fixed. If you select the current minute, Euphoric schedules the next minute instead of rejecting it as already passed.
