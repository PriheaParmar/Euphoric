## 🚀 Install

1. Download or clone this repository.
2. Open Chrome.
3. Go to `chrome://extensions`.
4. Turn on **Developer mode**.
5. Click **Load unpacked**.
6. Select the `chrome-extension` folder.

That’s it. Euphoric becomes your new tab.

---

## 🦋 Optional Butterfly Desktop Reminder

The extension works without this, but the floating desktop butterfly reminder needs the Windows companion.

1. Open:

```text
RUN_THIS_FOR_LOGO_BUTTERFLY_COMPANION_windows
```

2. Run:

```text
RESET_AND_INSTALL_COMPANION.bat
```

3. Restart Chrome.

---

## 🎧 Spotify Setup

Spotify needs your own Spotify Developer Client ID.

1. Create a Spotify Developer app.
2. Copy your **Client ID**.
3. Open Euphoric → Spotify.
4. Paste your Client ID.
5. Copy the redirect URL shown inside Euphoric.
6. Add that redirect URL in your Spotify app settings.
7. Connect again.

> Spotify playback controls may need an active Spotify device, and some controls may require Spotify Premium.

## 🔐 Permissions

Euphoric uses permissions for features only:

| Permission | Why |
|---|---|
| `storage` | Save tasks, settings, shortcuts |
| `alarms` | Reminder scheduling |
| `nativeMessaging` | Optional butterfly desktop companion |
| `identity` | Spotify login |
| `topSites` | Auto shortcuts |
| `history` | Recent website shortcuts |
