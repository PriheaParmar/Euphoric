Euphoric v6.8 Reminder Repair

Use this build because v6.3 optimized broke the reminder connection.

1. Open chrome://extensions
2. Remove old Euphoric versions
3. Load unpacked: LOAD_THIS_IN_CHROME_euphoric-extension-v6.8
4. Open RUN_THIS_FOR_LOGO_BUTTERFLY_COMPANION_windows
5. Double-click RESET_AND_INSTALL_COMPANION.bat
6. Close ALL Chrome windows completely
7. Reopen Chrome
8. Euphoric > Tasks > Test logo butterfly

If Test still does not work:
- Run test_companion_direct.bat first
- Then run check_native_host_install.bat
- Send the log files from this folder
