Euphoric v7.4

Full updated extension with:
- Auto shortcuts from most visited + recent websites
- topSites/history permissions added
- No manual add-shortcut UI
- Keeps Spotify, Focus, Tasks, Gita lessons, and butterfly reminder companion

Install:
1. Remove old Euphoric in chrome://extensions
2. Load unpacked: LOAD_THIS_IN_CHROME_euphoric-extension-v7.4
3. Run RESET_AND_INSTALL_COMPANION.bat from the companion folder if reminders need reconnecting
4. Restart Chrome

Important:
Chrome will ask for history/topSites permission because automatic shortcuts need it.
