# Privacy Notes

Euphoric stores tasks, focus settings, and shortcut preferences locally in the browser.

Automatic shortcuts use Chrome's `topSites` and `history` permissions to display recent/visited websites on the new tab. This data is processed locally by the extension.

Spotify integration connects directly to Spotify's Web API using Spotify login. Euphoric does not include a client secret.

The optional Windows companion receives reminder text from the extension to show the butterfly reminder animation on the desktop.


Euphoric stores a local random seed in browser storage so daily scenery and lessons can be randomized per browser. This stays local.
